## 🧮 Conversion Formulas

| From → To | Formula |
|---|---|
| Celsius → Fahrenheit | `(°C × 9/5) + 32` |
| Fahrenheit → Celsius | `(°F − 32) × 5/9` |
| Celsius → Kelvin | `°C + 273.15` |
| Kelvin → Celsius | `K − 273.15` |
| Fahrenheit → Kelvin | `(°F − 32) × 5/9 + 273.15` |
| Kelvin → Fahrenheit | `(K − 273.15) × 9/5 + 32` |

## 🛠️ Getting Started

### Prerequisites

Just a modern browser — no installs needed!

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/ankushkumar01/Temperature_Converter_Website.git

# 2. Navigate into the folder
cd Temperature_Converter_Website

# 3. Open in your browser
open index.html
# or double-click index.html in your file explorer
```

---

## 🧑‍💻 Usage

1. Enter a numeric temperature value in the input field
2. Select the **source unit** (e.g. Celsius)
3. Select the **target unit** (e.g. Fahrenheit)
4. Click **Convert** to see the result instantly 🎉
